# officialmain-leyu-notes

## 项目简介

本项目是一个用于归档和发布多个独立 HTML 页面的仓库。仓库内收录的内容以静态页面形式呈现，涵盖不同主题和用途。所有页面均为独立文件，不依赖外部站点或特定域名环境，方便直接浏览与保存。

本仓库不针对任何具体网站或服务，仅作为 HTML 页面的长期归档与展示用途。

## 目录结构

```
.
├── README.md          # 本说明文件
├── index.html         # 入口页面，可浏览所有归档页面列表
└── pages/             # 存放所有独立 HTML 页面
    ├── page-001.html
    ├── page-002.html
    └── ...
```

- `index.html`：提供导航和页面索引，方便快速查看所有归档内容。
- `pages/`：存放所有独立的 HTML 页面文件，文件名保持唯一性，便于引用与查看。

## 页面归档说明

- 每个页面均为自包含的 HTML 文件，无需额外资源或网络请求即可在浏览器中打开。
- 页面内容可能涉及笔记、工具、信息展示等不同类别，具体内容请以页面本身为准。
- 归档页面可能不定期更新或添加新页面，旧页面通常保留不变，以保持历史记录。

## 维护说明

- 本仓库由作者不定期维护，新增页面或修正内容时会在提交记录中说明。
- 欢迎通过 Issues 提出建议或报告页面问题，但不保证所有反馈都会被处理。
- 仓库内容仅供个人归档与分享，不构成任何形式的服务承诺或商业用途。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 使用任意浏览器打开 `index.html` 或 `pages/` 下的具体页面文件。
3. 也可直接通过 GitHub Pages（如已启用）在线浏览。
